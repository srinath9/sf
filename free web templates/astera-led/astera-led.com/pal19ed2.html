Wireless Pixel Tube

product code: AL1

Our innovative Pixel Tube is easily manageable using our remote control technology. It can be separated into multiple pixels or linked with our other lighting objects. The Pixel Tube can be inserted into design-objects like bars, tables etc. For large-scale event decoration, several pixel tubes can be programmed together to display complex ambience lighting.

Technical Specification

  • Illuminants:

    64 RGB LED chips + 64 white LED chips

  • LED RGB Power:

    16W

  • Beam Angle:

    180 degree

  • Pixel:

    16

  • Battery:

    Built-in Samsung battery for up to 24h operation

  • Charger Input:

    110-240VAC, 50/60Hz

  • Charger Output:

    48VDC, 0.5A

  • Wireless Module:

    868MHz receiver with up to 300m range (US version: 902-928MHz / 330yd)

  • DMX Capable:

    Wireless DMX with Astera Wireless DMX Transmitter ART3

  • Housing:

    Polycarbonate with metal end caps

  • Dimensions:

    Ø41.6mm x 1035mm (Ø1.64” x 40.75”)

  • Weight:

    0.950g

  • Environment:

    Indoor & outdoor (splash-proof)

  • Extras:

    Gamma correction, temperature compensation, color calibration

  • Included Accessory:

    48V, 0.5A Power supply unit

  • Optional Accessory:

    Metal holder-rail for hanging and screwing the AL1 to walls/ceilings

  • Suitable Applications:

    Under table lighting, effect lighting, furniture lighting, object illumination, built-in installations, event lighting
