Pixel Tube

código de producto: AL1

Nuestro innovador Pixel Tube es controlado fácilmente con nuestra exclusiva tecnología a control remoto. Se puede separar en múltiples píxeles o vincular a nuestros otros objetos de iluminación. El Pixel Tube se puede insertar en objetos diseñados como barras de bar, mesas, etc. Para la decoración de eventos a gran escala, varios Pixel Tubes se pueden programar en conjunto para mostrar una iluminación ambiente más compleja

Especificaciones técnicas

  • Iluminantes:

    chips de 64 LED RGB + chips de 64 LED blancos

  • Energía del LED RGB:

    16W

  • Ángulo de haz:

    180 grados

  • Pixel:

    16

  • Batería：

    Batería de litio manganeso – níquel - cobalto incorporada para 8-24h de uso

  • Voltaje de Funcionamiento：

    110-240V, 50/60Hz

  • Módulo Inalámbrico：

    Receptor 868 MHz con hasta 300m de alcance (versión EE.UU.: 902-928MHz / 330Yd)

  • Capacidad DMX:

    DMX inalámbrico con Astera Wireless DMX Transmitter ART3

  • Carcasa：

    Policarbonato con tapas de metal

  • Dimensiones：

    L100cm x H4cm x A4cm (L39.3 "x H1.6" x A1.6 ")/p>

  • Peso：

    1.000kg (2,20 libras)

  • Ambiente：

    Interior y exterior (a prueba de salpicaduras)

  • Extras：

    Corrección Gamma, compensación de temperatura, calibración del color

  • Accesorios Incluidos:

    Fuente de alimentación 48V, 0.5A

  • Accesorio opcional：

    Carril Metálico para colgar y atornillar el AL1 a las paredes / cielo raso

  • Aplicaciones adecuadas：

    Iluminación debajo de mesas, efecto de iluminación, iluminación de muebles, iluminación de objetos, incorporado en instalaciones, iluminación de eventos
