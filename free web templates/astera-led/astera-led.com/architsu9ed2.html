Itsu
Retail lighting for a modern world

A unique interior design installation was installed in a restaurant belonging to the Japanese Itsu chain in England. The LED fitting is inspired by a colorful butterfly flying along at the center of the ceiling. The installation manages to balance between being dynamic yet tranquil and provides an eye catcher to attract customers and a pleasant atmosphere while eating inside the venue.

This is the first site to feature an Itsu “Butterfly Bar”, and it features acrylic butterfly wings which are illuminated by a set of LED strips provided by Astera. Upon careful inspection, you can see that only half of the butterflies are hanging from the wall, with the other half being seen via a cleverly placed mirror. This project has caused Astera to start considering additional setups with acrylic light-medium for special projects and standard products.
