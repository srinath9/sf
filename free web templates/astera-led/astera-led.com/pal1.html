像素管

产品代码: AL1

我们创新的像素管可以通过遥控技术简易控制。它自身可以分成多个独立像素或与我们其它照明灯具连接使用。像素管可以安装在一些定制产品品内，如吧台、桌子等。大型的装饰照明项目可以通过编辑多个像素管获得各种灯光效果。

技术说明

  • 光源：

    64颗RGB 全彩LED灯+64颗白灯

  • RGB全彩 LED功率：

    16W

  • 光束角：

    180度

  • 像素：

    16个

  • 电池：

    三元材料电芯，可持续工作8-24小时

  • 工作电压：

    AC110-240V, 50/60Hz

  • 无线模块：

    868MHz接收范围达300米( US版本：902-928MHz/330码 )

  • DMX功能：

    可以通过Astera无线DMX发射器ART3传输无线DMX信号

  • 外壳：

    PC管，带金属端盖

  • 尺寸：

    L100cm x H4cm x W4cm (L39.3” x H1.6” x W1.6”)

  • 重量：

    1.000kg(2.20lb)

  • 适用环境：

    室内&室外（防溅式防水）

  • 其他特点：

    Gamma校正功能，温度补偿功能，颜色校准

  • 固定配件：

    48V, 0.5A充电器

  • 可选用配件：

    金属导轨，用于把产品悬挂和固定在墙上或建筑物顶部。

  • 应用范围：

    桌子底部照明，效果照明，家具照明，物件照明，内置式安装产品，活动照明

主要优点

  • 内置可充电池

  • 遥控功能

  • LED功率为16W

  • 无线DMX

  • 可放置在其他物件内使用

  • 180°光束角

配用遥控器

  • ARC1

  • ARC2

  • ART3

  • AsteraBox™

配件

  • 充电箱
