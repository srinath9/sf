Bañador de pared

código de producto: AL6 series

Nuestro Bañador de Pared Inalámbrico está diseñado para la iluminación de eventos y para la iluminación decorativa sin complicaciones. Basta con colocarlo al lado de una pared, encenderlo y el muro se bañará de colores.

Especificaciones técnicas

  • Iluminantes：

    AL6 S - 96 LEDs de tipo bala ( 24xR , 24xG , 24xB , 24xW ) = 12W potencia RGB AL6 -M 196 LEDs de tipo bala ( 48xR , 48xG , 48xB , 48xW ) = 24W potencia RGB AL6 L - 384 LEDs de tipo bala ( 96xR , 96xG , 96xB , 96xW ) = 48W potencia RGB

  • Ángulo de haz:

    15 grados

  • Difusor de la lente:

    Incluida, se pueden adaptar con facilidad

  • Batería:

    Batería de Li-Ion Polímero de 8-24h de funcionamiento

  • Voltaje:

    110 -240V , 50/60Hz

  • Módulo Inalámbrico：

    Receptor 868 MHz con hasta 300m de alcance ( versión EE.UU. : 902- 928MHz / 330Yd )

  • Capacidad DMX：

    DMX Over sockets integrados o W -DMX con ART3 ( transmisor inalámbrico DMX)

  • Carcasa：

    Carcasa de aluminio lineal y cubierta transparente de PMMA

  • Dimensiones；

    AL6 - S L25cm x H6.1cm x A6.1cm ( L9.8 " x H2.4 " x A2.4 " ) AL6 - M L50cm x H6.1cm x A6.1cm ( L19.6 " x H2.4 " AW2.4 " ) AL6 - L L100cm x H6.1cm x A6.1cm ( L39.2 " x H2.4 " x A2.4 " )

  • Peso：

    1 kg ( 2,2 libras ) 2 kg ( 4,4 libras ) 4 kg ( 8,8 libras )

  • Medio ambiente：

    interior y seco al aire libre

  • Extras：

    Corrección Gamma , compensación de temperatura , calibración de color

  • Accesorios incluidos：

    Cable para la fuente de alimentación incorporada, lente difusora

  • Aplicaciones adecuadas:

    Iluminación mesas / muebles, efectos de iluminación, iluminación de paredes, iluminación para la playa o árboles

Principales Beneficios

  • Batería recargable

  • Controlado a distancia

  • Potencia LED 12 - 48W

  • DMX inalámbrico

  • Funcionamiento sin parpadeo

  • Funcionamiento autónomo
