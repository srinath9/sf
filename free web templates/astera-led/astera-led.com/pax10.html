SpotMax™ 顶级射灯

产品代码: AX10

AX10作为我们X系列产品的代表作，是最新一代的无线照明灯具，是我们在无线射灯领域多年的技术积累成果。SpotMax™添加了琥珀色LED以取得更完美的灯光色彩饱和度。它可以用我们所有的遥控器，无线DMX发射机或第三方的W-DMX控制器进行控制。

技术说明

  • 光源：

    9 x RGBAW Philips LEDs

  • LED总功率：

    135W

  • 光束角：

    13度 (需配用Khatod透镜)

  • 电池：

    内置三星电池，可持续工作8-24小时

  • 工作电压：

    AC110-240V, 50/60Hz

  • 无线模块：

    868MHz接收范围达300米( US版本：902-928MHz/330码 )

  • DMX功能：

    通过Astera无线DMX发射器实现无线DMX功能, 或通过XLR适配器实现有线DMX功能

  • 控制面板：

    5键按键膜，3行蓝/白显示屏，可进行各种功能设定和DMX设置

  • 外壳：

    防水铸铁烤漆外壳(防水等级IP65)

  • 尺寸：

    L265mm x H310mm x W285mm

  • 重量：

    8.14kg

  • 其他特征：

    真色彩校准，自动添加白色，自动添加琥珀色，动态功率驱动，双智能天线，散热功能强大的独特外壳

  • 适用场合：

    舞台灯光，剧场灯光，射灯，活动照明，夜店照明，户外照明

主要优点

  • 可充电电池

  • RF遥控控制+Wireless DMX

  • 超高亮135W LED

  • 无线DMX

  • 防水外壳

  • 无闪烁操作

  • 2米距离的亮度高达10370lux

  • CRI(显色指数) ≤ 92

配用遥控器

  • ARC1

  • ARC2

  • ART3

  • AsteraBox™

  • CRMX Nova TX
