Lightdrop™ 光滴

产品代码: AX3

我们的光滴Lightdrop™产品是活动照明的先进射灯。它是目前市场上最简洁的无线射灯，它可以吸附在墙上，天花板或直接放在PE产品内使用。

技术说明

  • 产品代码:

    AX3
    AX3-CRMX (带CRMX DMX 功能)

  • LED功率:

    15W (1 x RGBW CREE LED Chip)

  • 光通量:

    红光：160lm
    绿光：170 lm
    蓝光：56lm
    白光：390lm( 色温4000K)

  • 亮照度:

    红光：475 lx
    绿光：590lx
    蓝光：930lx
    4000K 白光：1450lx (色温 4000K)

  • 频闪频率范围:

    0-25Hz

  • 光角度:

    13°（配用Khatod透镜）

  • 电池:

    内置三星电池

  • 电池寿命:

    / 300次充放 (300次充放后电量降为

  • 电池工作时间:

    5-20 小时

  • LED 寿命:

    50,000 小时

  • 充电时间

    5-7 小时 (充电时间的长短受环境温度影响)

  • 工作电压:

    输入110-240V, 50-60Hz

  • 充电电流:

    2.5A at 5V

  • 无线模块:

    868MHz 接收模块（美规：902-928MHz）

  • 摇控距离:

    300米（330码)

  • DMX 介面:

    无线DMX （配用 ART3 DMX讯号发射器）

  • 可选用的DMX:

    CRMX无线DMX,
    同时可支持常规无线DMX

  • 外壳:

    导热尼龙

  • 防水等级:

    IP65

  • 使用湿度:

  • 工作温度:

    0°-40°C (32°-104°F)

  • 重量:

    0.430kg (0.9lbs)

  • 尺寸:

    L119mm x H59mm x W110mm
    L5.1” x H2.3” x W4.3”

  • 使用环境:

    户内或户外

  • 特征:

    / 真色彩校准功能，温度补偿功能
    智能功率自动调节功能

  • 标准配件:

    / 充电器，散光罩，吊环，支架,
    橡胶脚（4个），充电口堵盖

  • 可选配件:

    17°x46° 17°x46°洗墙效果散光片 ， 32° 投光效果散光片

主要优点

  • / 亮度是AL3-M的两倍

  • 内置三星可充电电充池

  • 带摇控功能

  • 15W 科锐高亮LED

  • CRMX 无线DMX

  • 底部带磁铁
