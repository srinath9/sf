This image (or all images in this category) uses inside descriptions or captions in a specific script or language and should be converted to a language neutral form. This would allow its use in all Wikimedia projects and, more importantly, all Wikimedia languages. Place {{Convert to international}} on an image to add it to this category.
Once a language neutral version exists please add it to Category:Language-neutral diagrams.
