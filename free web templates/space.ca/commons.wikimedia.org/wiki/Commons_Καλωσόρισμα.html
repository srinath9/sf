Τι είναι τα Wikimedia Commons;[edit]

Τα Wikimedia Commons είναι μια αποθήκη πολυμέσων που δημιουργήθηκε, εμπλουτίζεται και συντηρείται όχι από έμμισθους καλλιτέχνες, αλλά από εθελοντές. Το όνομά τους προέρχεται από το όνομα "Wikimedia", που αναφέρεται στο όνομα του Ιδρύματος Wikimedia που διαχειρίζεται όλα τα εγχειρήματα Wikimedia και από τον πληθυντικό της λέξης "κοινός", καθώς τα περιεχόμενά τους τα μοιράζονται διάφορα είδη εγχειρημάτων Wikimedia σε διάφορες γλώσσες. Έτσι, αποτελεί ένα κοινό, κεντρικό αποθετήριο ελεύθερων φωτογραφιών, εικόνων, διαγραμμάτων, μουσικής, ηχογραφημένου λόγου, βίντεο και όλων των ειδών αρχεία που είναι χρήσιμα στα εγχειρήματα Wikimedia.

Τα Wikimedia Commons χρησιμοποιούν την τεχνολογία wiki που ίσως ήδη γνωρίζετε από τη Βικιπαίδεια, κι έτσι ο καθένας μπορεί να τα επεξεργαστεί εύκολα και χωρίς τεχνικές γνώσεις, κατευθείαν από τον browser του. Αντίθετα με τα αρχεία που επιφορτώνονται σε ένα συγκεκριμένο εγχείρημα Wikimedia, τα αρχεία στα Wikimedia Commons μπορούν να χρησιμοποιηθούν σε όλα τα εγχειρήματα Wikimedia χωρίς να χρειάζεται να επιφορτωθούν ξεχωριστά σε κάθε ένα από αυτά.

Τα Wikimedia Commons δημιουργήθηκαν στις 7 Σεπτεμβρίου 2004, ξεπέρασαν το ένα εκατομμύριο επιφορτωμένα αρχεία στις 30 Νοεμβρίου του 2006 και σήμερα φιλοξενούν 22,743,139 αρχεία και 117,360 συλλογές. Περισσότερα στοιχεία για τα Wikimedia Commons μπορούν να βρεθούν στη Γενική αποποίηση, και τη σχετική σελίδα στο Meta-wiki.

Αντίθετα με τα συνηθισμένα αποθετήρια πολυμέσων, τα Wikimedia Commons είναι ελεύθερα. Ο καθένας είναι ελεύθερος να αντιγράψει, να χρησιμοποιήσει και να τροποποιήσει οποιοδήποτε από τα αρχεία που φιλοξενούμε, αρκεί να αναφέρεται η πηγή και οι δημιουργοί, και να συνεχίζουν να ισχύουν αυτές οι ελευθερίες για όλους. Η βάση δεδομένων των Wikimedia Commons και τα κείμενα που περιέχονται σε αυτά διατίθενται με την άδεια GNU Free Documentation License. Οι όροι χρήσης κάθε αρχείου μπορούν να βρεθούν στη σελίδα περιγραφής του. Περισσότερα πάνω στους όρους χρήσης...
Συμμετοχή

Μπορείτε να βελτιώσετε τα Wikimedia Commons κατά πολύ αν συνεισφέρετε με αυτό που μπορείτε να κάνετε καλύτερα:

Συνεισφέρετε τα έργα σας[edit]

Αν τα καταφέρνετε με τη φωτογραφία, μη διστάσετε να συνεισφέρετε τις φωτογραφίες σας. Αν είστε καλός σχεδιαστής, δείτε ποιά διαγράμματα και σχήματα χρειάζονται επειγόντως. Αν είστε καλός μουσικός ή ηθοποιός, μη διστάσετε να επιφορτώσετε δικές σας ηχογραφήσεις ή ερμηνείες ελεύθερων έργων.

Συνεισφέρετε τις ικανότητές σας[edit]

Η επιφόρτωση των δημιουργιών σας δεν είναι ο μοναδικός τρόπος συνεισφοράς. Υπάρχει πολλή ακόμα και σημαντική δουλειά που μπορείτε να κάνετε:

  • Μεταφράστε τα κείμενα βοήθειας σε κάποια γλώσσα εκτός από τα Αγγλικά.
  • Βελτιώστε εικόνες.
  • Ταυτοποιήστε άγνωστα θέματα.
  • Συνεισφέρετε τις νομικές σας γνώσεις στις ερωτήσεις περί πνευματικών δικαιωμάτων και τις σελίδες συζητήσεων για Εικόνες υποψήφιες για διαγραφή.

Συνεισφέρετε το χρόνο σας[edit]

Ακόμα και αν δεν σας τραβά η δημιουργία έργων ή το γράψιμο, μπορείτε να βοηθήσετε. Αν σας αρέσει να δημιουργείτε τάξη από το χάος, τότε σίγουρα σας χρειαζόμαστε!

  • Κάθε φορά που βλέπετε ένα αρχείο που δεν αναφέρει την άδεια χρήσης ή την πηγή του, σημάνετέ το με το πρότυπο {{subst:nsd}}.
  • Βάλτε τα "ορφανά" αρχεία σε κατηγορίες ή πινακοθήκες.
  • Βάλτε τις Εικόνες εκτός κατηγορίας σε κατηγορίες.
  • Πηγές ελεύθερων αρχείων περιμένουν να επιφορτωθούν εδώ.
  • Προτείνετε εικόνες ή ψηφίστε στις Υποψήφιες Αξιόλογες Εικόνες (για συνδεδεμένους χρήστες).
  • Γίνετε εθελοντής στο FlickrLickr, που επιφορτώνει ελεύθερες εικόνες από τον ιστότοπο flickr.
Εγγραφή

Για να μπορείτε να επιφορτώνετε αρχεία στα Wikimedia Commons, πρέπει να έχετε ένα λογαρισμό χρήστη και να είστε συνδεδεμένοι. Μπορείτε να δημιουργήσετε ένα λογαριασμό, καθώς και να συνδεθείτε, στο σύνδεσμο "Δημιουργία Λογαριασμού / είσοδος" στην πάνω δεξιά γωνία. Το όνομα χρήστη που θα διαλέξετε θα εμφανίζεται σε όλες τις επιφορτώσεις και επεξεργασίες που θα κάνετε. Αν απλά θέλετε να επεξεργαστείτε σελίδες και όχι εικόνες, δεν χρειάζεται να είστε συνδεδεμένοι (αν και προτείνεται). Αν έχετε κάνει ενοποίηση λογαριασμών, τότε έχετε ήδη λογαριασμό στα Wikimedia Commons.

Πρώτα βήματα

Οι οδηγοί Πρώτα βήματα και Συχνές ερωτήσεις θα σας βοηθήσουν μετά την εγγραφή σας. Εξηγούν πως να ρυθμίσετε τις προτιμήσεις χρήστη (για παράδειγμα τη γλώσσα στην οποία θα εμφανίζονται τα διάφορα μηνύματα συστήματος), πως να επιφορτώνετε αρχεία, και τη βασική πολιτική αδειοδότησης. Δεν απαιτούνται τεχνικές γνώσεις για να συνεισφέρετε εδώ. Μη διστάζετε να συνεισφέρετε και υποθέστε καλή πρόθεση για τους άλλους συνεισφέροντες. Βρισκόμαστε σε ένα wiki - η συμμετοχή είναι εύκολη.

Περισσότερες πληροφορίες θα βρείτε στην Πύλη Κοινότητας. Μπορείτε να κάνετε ερωτήσεις για οποιοδήποτε πρόβλημα στην Αγορά ή στο IRC κανάλι #wikimedia-commons.

Τα αρχεία των Κοινών ταξινομούνται σε κατηγορίες και συλλογές εικόνων. Μια συνοπτική εικόνα των κατηγοριών θα δείτε στην Κύρια Σελίδα.

Επιπλέον δυνατότητες και εργαλεία

Παρόλα αυτά, αν ακόμα οι κανόνες σας φαίνονται περίπλοκοι ή θέλετε να επιφορτώσετε μεγάλο αριθμό έργων, μπορείτε να χρησιμοποιήσετε την υπηρεσία επιφόρτωσης αρχείων, όπου πιο πεπειραμένοι χρήστες θα αναλάβουν την επιφόρτωση του υλικού σας. Αν δεν χρειάζεστε βοήθεια αλλά θέλετε απλά να επιφορτώσετε μεγάλο αριθμό αρχείων, μπορείτε να χρησιμοποιήσετε το λογισμικό Commonist. Αν χρειάζεστε πιο συγκεκριμένες πληροφορίες για εργαλεία που θα σας βοηθήσουν να επιφορτώσετε και να επεξεργαστείτε αρχεία, δείτε τη σελίδα Commons:Software και Commons:Tools.

Ελπίζουμε ότι η συμμετοχή σας στα Wikimedia Commons θα είναι μια ευχάριστη και δημιουργική εμπειρία.
