English: Intentionally empty templates used as tags. Useful for invisibly tagging images or categories which then can be intersected with CatScan. The templates should be protected in order so they remain empty.
See more here

This template is empty: it does not take any inputs and does not produce any output. It is used for invisibly tagging pages which then can be intersected with other tags or categories using CatScan tools or database queries. It is meant to be used in similar way as hidden maintenance category, except that it is even more hidden. This template should be protected so it remains empty.

Usage

{{Empty tag templates}}

The template takes no parameters.

Additional information

The template is intended to be used in the following namespaces: the Template namespace

The template is intended to be used by the following user groups: no user group specified

{{Books published before 1923}} {{Book}} If book was published before 1923 All Adding {{PD-1923}} license. Uses:
  • CatScan2 query will find book templates which use {{PD-old}} (or related templates) but no {{PD-1923}}
{{Works of authors who died less than 65 years ago}} {{Creator}} If person died less than 65 years ago All Incompatible with {{PD-old}} licenses which require person to be dead for 70 years. Added 5 year margin of error. Use CanScan2 to find all the files using creator templates of authors who died less than 70 years ago and incorrectly using {{PD-old}} license.
{{Works of authors who died more than 100 years ago}} {{Creator}} If person died more than 100 years ago All Adding {{PD-old-100}} license. Make sure that work did not have multiple authors.
{{Works of authors who died more than 70 years ago}} {{Creator}} If person died between 70 and 100 years ago All Adding {{PD-old-auto}} or {{PD-old-70}} license. Make sure that work did not have multiple authors.
{{Creator template no image}} {{Creator}} Creator template does not have an image File Used to add images to creator templates. For example, use CatScan2 to find all the creator templates without images used within files which are in the Category:Self-portraits.
{{Creator template: ignore this selfportrait}} added to individual images In case this image is a self portrait and one of the authors has a creator template with no image, but this image depicts the other author, than this tag added to an image will exclude it from the results of the query in the line above.
