To include a link to an audio file in an article like for pronunciations, you use a template. But not all Wikipedias use the same template. This page is to have an overview of those templates for the several Wikipedias.

These templates are in the first place for adding the pronunciation to an article by a user who does not know the language of that Wikipedia.
So do not use large templates or templates where a user needs to fill in a description or so. The only thing that a user needs to change is the name of the audio file.

BE.wikipedia.org[edit]

general audio template

  • w:be:Шаблён:Праслухаць
    • {{Праслухаць|name of file on commons.ogg}}

BG.wikipedia.org[edit]

general audio template

  • w:bg:Шаблон:Звук
    • {{звук|name of file on commons.ogg|what is pronounced}}
      • Example: {{звук|En-us-George Walker Bush.ogg|George Walker Bush}}

CA.wikipedia.org[edit]

  • w:ca:Plantilla:Audio
    • {{Audio|name_of_file_on_commons.ogg|escolteu}}

CS.wikipedia.org[edit]

  • w:cs:Template:Audio
    • {{Audio|filename.ogg|what is pronounced}}
      • Example: {{Audio|En-us-George Walker Bush.ogg|George Walker Bush}}
  • w:cs:Template:Poslech (usually not suitable for pronunciation, though)
    • {{Poslech|nazev_souboru=''filename.ogg''|titulek=''title''|popis=''description''}}

DA.wikipedia.org[edit]

  • w:da:Skabelon:Audio
    • {{Audio|name_of_file_on_commons.ogg|hør}}

DE.wikipedia.org[edit]

  • w:de:Vorlage:Audio
    • {{Audio|name_of_file_on_commons.ogg|anhören}}

EN.wikipedia.org[edit]

  • w:en:Template:Audio
    • {{Audio|name_of_file_on_commons.ogg|listen}}
  • w:en:Template:Audio-nohelp
    • {{Audio-nohelp|name_of_file_on_commons.ogg|listen}}
  • w:en:Template:Listen
    • {{listen|filename=name_of_file_on_commons.ogg|title=Title of this file|description=The description of this file|format=[[Ogg]]}}
  • w:en:Multi-listen start, w:en:Multi-listen item, and w:en:Multi-listen end
    • {{multi-listen start}}
      {{multi-listen item|filename=name_of_file_on_commons.ogg|title=Title of this file|description=The description of this file|format=[[Ogg]]}}
      {{multi-listen item|filename=name_of_file_on_commons.ogg|title=Title of this file|description=The description of this file|format=[[Ogg]]}}
      {{multi-listen item|filename=name_of_file_on_commons.ogg|title=Title of this file|description=The description of this file|format=[[Ogg]]}}
      {{multi-listen end}}

Es.Wikipedia.org[edit]

  • w:es:Plantilla:Audio
    • {{audio|1=filename on wikipedia.ogg|2=Listen(or pronunciation or whatever you want to appear to be clicked on.)}}
  • w:es:plantilla:escuchar
    • {{escuchar|name of file on commons.ogg|tit=title of this file|des= The description of this file|format=ogg}}

ET.wikipedia.org[edit]

  • w:et:Mall:Kuula (listen)

FR.wikipedia.org[edit]

  • w:fr:Modèle:Prononciation
    • {{Prononciation|name_of_file_on_commons.ogg}}

    • {{Son simple|description of the file|name_of_file_on_commons.ogg}}
  • w:fr:Modèle:Son more generic, bigger, stamp.
    • {{Son|description of the file|name_of_file_on_commons.ogg}}

IS.wikipedia.org[edit]

  • w:is:Snið:Hljóð

special for pronunciations

  • w:nl:Sjabloon:Audio

NO.wikipedia.org[edit]

  • w:no:Mal:Lydfil
    • {{Lydfil|name_of_file_on_commons.ogg|uttale}}
      • Use this template for pronunciation recordings; uttale = pronunciation
    • {{Lydfil|name_of_file_on_commons.ogg|hør}}
      • use this template for non-pronunciation files; hør = listen

PT.wikipedia.org[edit]

  • w:pt:Predefinição:Áudio nu (best for sounds of pronunciation)
    • {{Áudio nu|filename|title}}
  • Titles / words
play = tocar
listen = escutar
to hear = ouvir

When there is no known template[edit]

To put on the talk page of the article

Hi, I have here a related audio-file for this article; *[[Media:.... but do not know the audio template for pronunciations of this Wikipedia. Please place this in the article. To be able to include this directly in the articles please let know how your audio template works. You can do this by adding it to [[commons:List of the different audio-templates for the Wikipedias]]. Greetings; [[w: ]]
