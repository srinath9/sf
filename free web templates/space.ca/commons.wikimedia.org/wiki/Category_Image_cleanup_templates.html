This category is used for categorizing templates.
It should not contain media files or pages outside the template namespace.

This category contains templates used to mark images as requiring some type of cleanup or editing. See Commons:Images for cleanup for details on how they should be used. Note that any new templates added to this category should use the "<noinclude> ... </noinclude>" markup to prevent all images bearing the template from being added to this category.
