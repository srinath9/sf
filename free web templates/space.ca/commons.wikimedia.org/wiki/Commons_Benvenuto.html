Questa WikiGuida ti può aiutare a compiere i primi passi nel mondo di Commons

Wikimedia Commons (o più semplicemente "Commons") è una repository di file multimediali che consente a chiunque di accedere a contenuti educativi nel pubblico dominio e con licenza libera (immagini, suoni, videoclip), nella propria lingua. Funge da comune archivio per i vari progetti della Wikimedia Foundation, ma non è necessario appartenere ad uno di questi progetti per usare i file multimediali ospitati qui. La repository è creata e gestita da volontari, non da artisti pagati. Lo scopo di Commons è definito nelle pagine degli scopi del progetto.

Wikimedia Commons usa la stessa wiki-tecnologia di Wikipedia e chiunque può modificare le pagine. A differenza dei file multimediali caricati in altri progetti, i file caricati su Wikimedia Commons possono essere usati direttamente nella pagine di tutti i progetti Wikimedia, senza la necessità di caricarli seperatamente.

Lanciato il 7 settembre 2004, Wikimedia Commons ha raggiunto 1.000.000 di file caricati il 30 novembre 2006 e attualmente contiene 22,743,139 file e 117,360 gallerie multimediali. Maggiori informazioni riguardo il progetto Wikimedia Commons si trovano nelle avvertenze generali, nella pagina di Wikipedia su Wikimedia Commons e nella pagina dedicata su Meta-wiki.

A differenza delle tradizionali repository di file multimediali, Wikimedia Commons è libero. A chiunque è permesso copiare, usare e modificare liberamente qualsiasi file multimediale a patto che venga indicata la fonte e che le opere vengano attribuite ai rispettivi autori; inoltre, gli utenti devono pubblicare le loro copie/miglioramenti con lo stesso grado di libertà degli altri. Lo stesso Wikimedia Commons e i testi contenuti sono pubblicati con licenza Creative Commons Attribuzione-Condividi allo stesso modo. La licenza d'uso di ogni singolo file multimediale può essere visionata nella propria pagina di descrizione. Maggiori informazioni sul riutilizzo dei file possono essere trovate in Commons:Riuso del contenuto al di fuori di Wikimedia e Commons:Primi passi/Riutilizzazione.

Partecipare

Puoi migliorare Wikimedia Commons contribuendo con ciò che sai fare meglio:

Contribuisci con le tue opere[edit]

Se sei un buon fotografo non esitare a contribuire con le tue immagini migliori. Se sei un buon disegnatore, controlla quali diagrammi e animazioni sono richieste. Se sei un buon musicista o artista di teatro o cinematografia, non esitare a pubblicare con licenza libera le registrazioni delle tue performance.

Contribuisci con le tue abilità[edit]

In ogni caso, non è necessario solo caricare file. Ci sono cose altrettanto importanti da fare:

  • tradurre i testi di aiuto in una lingua diversa dall'inglese;
  • migliorare le immagini;
  • identificare soggetti sconosciuti;
  • contribuire con le tue conoscenze legali nelle questioni di copyright e nelle richieste di cancellazione.

Contribuisci con il tuo tempo[edit]

Non hai bisogno di essere un artista o un bravo scrittore di testi di aiuto. Se ti senti portato a creare ordine dal caos, abbiamo bisogno di te!

  • Ogni volta che vedi un file con informazioni carenti o incomplete sulla licenza o sulla fonte, segnalala inserendo nella pagina di descrizione il codice {{subst:nsd}}.
  • Dai almeno una categoria alle pagine non categorizzate.
  • Sposta i file nelle sottocategorie appropriate
  • Le risorse di file multimediali liberi ti aspettano per essere incluse in questo progetto.
  • Nomina o vota per le immagini in vetrina (registrazione richiesta).
Registrazione

Per poter caricare file su Wikimedia Commons è necessario effettuare il login. Puoi registrarti cliccando sul collegamento "Log in / create account" presente nell'angolo in alto a destra e inserendo un nome utente che sarà usato per tutti i tuoi caricamenti di file e modifiche di pagine. Comunque, se vuoi solo modificare le pagine non hai bisogno di registrarti (anche se ti incoraggiamo a farlo). Se hai il login unificato, sei già registrato anche in Commons.

Tutoriali per i primi passi

Le pagine primi passi e le FAQ ti aiuteranno molto dopo la registrazione. Spiegano come personalizzare l'interfaccia (per esempio la lingua), come caricare file e le basi della nostra politica sul licenziamento. Non è necessario avere particolari abilità tecniche per contribuire qui. Sii audace nel contribuire qui e presumi la buona fede nelle intenzioni degli altri. Questa è una wiki - ed è veramente semplice.

Maggiori informazioni sono disponibili nel Portale comunità. Puoi porre delle domande al Bar o nel canale IRC #wikimedia-commons.

I file su Wikimedia Commons sono organizzati in categorie e gallerie. Un elenco delle principali categorie è disponibile nella Pagina principale.

Servizi aggiuntivi e software

Se vuoi caricare un gran numero di immagini, il programma Commonist ti sarà utile. Se hai bisogno di informazioni specifiche sui tool disponibili per vedere o modificare il nostro contenuto, vedi Commons:Software.

Buon divertimento!
