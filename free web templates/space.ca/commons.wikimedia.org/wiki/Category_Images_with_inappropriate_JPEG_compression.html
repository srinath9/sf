This category is used for administration or maintenance of Wikimedia Commons.
Files are placed in this category by adding the template {{BadJPEG}} to the file's description page.
Please do not add this category manually.
Maintenance: Search for misplaced files using CatScan (This will not work on categories shared by many templates.)

These images were uploaded in the JPEG format even though they contain non-photographic information. It is suggested that the original sources be revisited and that a PNG or SVG file without compression artifacts be uploaded in their places. For more information, see Commons:Preparing images for upload.

Computer and video game screenshots should be tagged with {{Convert to PNG}}.

Badly compressed jpeg files should be tagged with {{Low quality}}.
