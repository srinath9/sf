This category is used for administration or maintenance of Wikimedia Commons.
Files are placed in this category by adding the template {{Remove border}} to the file's description page.
Please do not add this category manually.
Maintenance: Search for misplaced files using CatScan (This will not work on categories shared by many templates.)

Images in this category have unnecessary borders that need to be removed. Where borders are desired they should usually be added with article markup such as table markup. If cropping a JPEG, consider using a lossless cropping tool such as jpegtran or Irfan View (Shift-J). You can also use Cropbot and the web-based CropTool (which supports lossless cropping) to trim images.

See also category: Images with intentional borders
