Images in this category are blurry, unfixably too light/dark, or may not sufficiently demonstrate the subject of the picture. If a higher-quality image is available please consider replacing them; otherwise replacements (under a free license) should be found or provided.

Please see Category:Image cleanup templates for templates used to mark images that only need a clean-up. For more help, see Commons:Images for cleanup#Low quality pictures.

Files are placed in this category by adding the template {{low quality}} to the file's description page.
Please do not add this category manually.
Maintenance: Search for misplaced files using CatScan (This will not work on categories shared by many templates.)
