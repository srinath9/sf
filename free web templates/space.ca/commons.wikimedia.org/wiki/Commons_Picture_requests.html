This page is a forum for requesting images from other Commons users for pages in Wikimedia projects, including Wikipedia articles. Purge this page

Adding a request[edit]

  • First try to find an image on your own (see help on public domain image resources)
  • Add your request to a subcaption (create a subcaption if necessary. Any language OK, English preferred) in the request page:
    • One request for one image only.
    • Link to a page where you want the image to be placed (any Wikipedia article / Wikimedia project page OK).
    • If possible, include an external link to a picture about the request.
    • Sign your request in case there are questions (use ~~~~).

Upload requested image[edit]

  • Upload the image, provide licensing information, source, and description.
  • Either
    • Add a line here that you uploaded the image, or
    • Remove the request from this list.
