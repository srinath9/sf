Category:Alphabet images that should use vector graphics

From Wikimedia Commons, the free media repository
Jump to: navigation, search

Images composed of simple shapes, lines, and letters such as those below should be recreated using vector graphics as an SVG file. These have a number of advantages such as making it easier for subsequent editors to edit them, enabling arbitrary scaling at high quality, and sharp high-resolution renderings for print versions. See Commons:Images for cleanup for more information.

This is a subcategory of Category:Images that should use vector graphics. You can add images to this category using the template {{Convert to SVG|alphabet}}.
